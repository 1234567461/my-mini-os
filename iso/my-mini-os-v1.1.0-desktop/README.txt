My Mini OS v1.1.0-desktop ISO镜像
===================================

版本: desktop
构建日期: 2026-06-25

使用方法:
1. 将ISO镜像写入U盘或光盘
2. 从U盘或光盘启动
3. 系统将自动进入对应的模式

版本说明:
- terminal: 终端版本 - 启动后直接进入命令行Shell
- setup: 安装向导版本 - 启动后直接进入Setup TUI安装向导
- desktop: 桌面版本 - 启动后直接进入GUI图形桌面
